# 🎓 Student Report Card Management System

A simple and interactive web-based application for managing student records and generating report cards. This project demonstrates the implementation of **Linear Search** and **Bubble Sort** algorithms using HTML, CSS, and JavaScript.

---

## 📌 Project Overview

The Student Report Card Management System allows users to:

- Add student information
- Store student records
- Generate individual report cards
- Display all student results
- Sort student records using Bubble Sort
- Search student records using Linear Search
- Delete all stored student records

This project was developed as part of the **Design and Analysis of Algorithms (DAA)** course to demonstrate the practical implementation of searching and sorting algorithms.

## 📚 Algorithms Used

### 🔍 Linear Search

Linear Search is used to find a student record by checking each record one by one until the required student is found.

**Time Complexity**

| Best Case | Average Case | Worst Case |
|-----------|--------------|------------|
| O(1) | O(n) | O(n) |

---

### 🔃 Bubble Sort

Bubble Sort is used to arrange student records in ascending order based on marks.

**Time Complexity**

| Best Case | Average Case | Worst Case |
|-----------|--------------|------------|
| O(n) | O(n²) | O(n²) |

## ⚙️ How It Works

### Step 1
Click **Add Student Details**.

### Step 2
Enter:

- Student Name
- Roll Number
- Subject Marks

### Step 3
Save the student information.

### Step 4
Generate an individual report card.

### Step 5
Display all stored student records.

### Step 6
Search a specific student using Linear Search.

### Step 7
Sort students based on marks using Bubble Sort.

### Step 8
Clear all records whenever required.
